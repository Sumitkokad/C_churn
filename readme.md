## 📌 Project Overview

Customer churn is a critical business problem where customers stop using a company's service. This project builds a **Machine Learning-based prediction system** to identify whether a customer is likely to churn or not 
Churn means the customer stops using the comapany serivce .

💡 By predicting churn in advance, companies can:
- Reduce revenue loss  
- Improve customer satisfaction  
- Take proactive retention actions  
- A customer stops using a company’s service is known as churn .

---

## 🎯 Problem Statement

In real-world businesses, losing customers directly impacts profit. However, it is difficult to manually identify which customers are likely to leave.

👉 This project solves that problem by:
- Analyzing historical customer data  
- Learning patterns of churn behavior  
- Predicting future churn using ML models

This project follows a complete **End-to-End Machine Learning Pipeline**:
